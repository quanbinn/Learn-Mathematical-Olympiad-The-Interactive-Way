# 问题7: 从相同的（所需的）数字中减去给定的两个数字，从而使剩余数的比值等于给定比例。

## 必要条件: 第二个数字必须小于第一个数字的给定分数比另外一个数字的给定分数大的给定的差值。

> 丢番图解法如下：
>  
- 给定数字是 100, 20，给定的比例 3:1。
- 所需数是 x。 因此 
-        x - 20 = 3 (x - 100)
-             x = 140
