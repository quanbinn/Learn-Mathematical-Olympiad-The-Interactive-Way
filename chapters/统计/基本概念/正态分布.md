```python
import numpy
import matplotlib.pyplot as plt

sampleset = numpy.random.normal(0.6, 0.1, 100) # 第一个argument为平均值，第二个argument为标准差，第三个为自动生成的样本的数量
print(sampleset)

plt.hist(sampleset, 100)  # 第二个argument代表的是柱状图的个数，即正态分布的总样本数会平均分配在这些柱状图的区间内
plt.show()
```
