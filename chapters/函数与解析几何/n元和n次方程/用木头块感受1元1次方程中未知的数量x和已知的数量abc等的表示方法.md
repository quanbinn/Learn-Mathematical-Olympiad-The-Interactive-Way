# 用木头块感受1元1次方程中未知的数量x和已知的数量abc等的表示方法

## 开始做实体实验

### x + a = b => x + a - a = b - a => x = b - a
### x + 5 = 9 => x + 5 - 5 = 9 - 5 => x = 9 - 5 => x = 4
### x - 7 = 4 => x -7 + 7 = 4 + 7 => x = 4 + 7 => x = 11
### 9 - x = 4 => 9 - x + x = 4 + x => 9 = 4 + x => x + 4 = 9 => x + 4 - 4 = 9 - 4 => x = 5
