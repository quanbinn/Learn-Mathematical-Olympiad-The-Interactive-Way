### y = -2x
```python
import numpy as np
import matplotlib.pyplot as plt
plt.gca().set_aspect( 1 ) 

x = np.linspace(start = -5, stop = 5, num = 6)
print(x)
y = -2*x
print(y)

plt.scatter(x, y) # plt.plot(x,y)
plt.show()
```
