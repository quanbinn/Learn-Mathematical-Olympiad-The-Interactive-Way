### y = x<sup>2</sup>- x - 2 的轮廓与x轴的交点就是y = x<sup>2</sup>的轮廓和y = x + 2的轮廓的交点
```python
import numpy as np
import matplotlib.pyplot as plt
plt.gca().set_aspect( 1 ) 

x = np.linspace(-3,3,35) # print(x)
y = x**2 - x - 2 # print(y)
plt.scatter(x,y)

y = x**2
plt.scatter(x,y)

y = x + 2 
plt.scatter(x,y)

y = x*0
plt.scatter(x,y)

plt.show()
```
