# 去括号法则（使用推演法则。。。。。）

## 本质意义: 
## 括号前面是加号时，去掉括号，括号内的所有加号和减号不变；
## 括号前面是减号时，去掉括号，括号内的所有加号变减号，所有减号变加号。

### 注明：括号即 ( )，指的是一对括号，包括前括号 ( 和后括号 ) 。

## 开始做实体实验

### 符号描述：
### 例如： 2 * (1 + 3) = 2 * 1 + 2 * 3 = 8
