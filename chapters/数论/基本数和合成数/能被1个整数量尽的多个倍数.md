# 能被1个整数量尽的多个倍数

## 本质意义：能被某个整数a量尽的无限多个整数被称为这个整数的倍数的集合，倍数集合中有无限多个数，形成一个公差为这个整数的等差数列 {a, 2a, 3a, ..., (n-1)a, na}。 

## 开始做实体实验

### A＝6＝2<sup>1</sup>×3<sup>1</sup>
- 1A = 6*1 = 6 
- 2A = 6*2 = 12 
- 3A = 6*3 = 18 
- 4A = 6*4 = 24 
- 5A = 6*5 = 30 
- 6A = 6*6 = 36 
- 7A = 6*7 = 42 
- 8A = 6*8 = 48 
- 9A = 6*9 = 54 
- 10A = 6*10 = 60

### B＝10＝2<sup>1</sup>×5<sup>1</sup>
- 1B = 10*1 = 10
- 2B = 10*2 = 20 
- 3B = 10*3 = 30 
- 4B = 10*4 = 40 
- 5B = 10*5 = 50 
- 6B = 10*6 = 60
