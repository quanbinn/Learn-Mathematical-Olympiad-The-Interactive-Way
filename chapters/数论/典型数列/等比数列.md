# 等比数列
### 对于数列{a<sub>n</sub>}，已知第一项为a<sub>1</sub>=a，公比为r(common ratio)。
### => {a<sub>n</sub>} =  {a, ar, ar<sup>2</sup>, ar<sup>3</sup>, ... , ar<sup>n-1</sup>}
### => 通项公式: a<sub>n</sub> = ar<sup>n-1</sup>
