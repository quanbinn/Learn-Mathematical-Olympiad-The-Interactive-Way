# 等差数列

### 对于数列{a<sub>n</sub>}，已知第一项为a<sub>1</sub>，公差为d。
### => {a<sub>n</sub>} =  {a<sub>1</sub>, a+d, a+2d, ... , a+(n-1)d, ...}
### => 通项公式: a<sub>n</sub> = a<sub>1</sub> + (n-1)*d 

## 开始做实体实验

###  首项为1的系列递增数列：
#### 数列：{a<sub>n</sub>} =  {1，2，3，4，5，6，... }
#### => 公差：d = 1,  通项公式: a<sub>n</sub>=1+(n-1)*1=1+n-1=n
